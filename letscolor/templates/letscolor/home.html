About This Coloring Book

"Color Book is a website in which kids can color the images online.

The app was made as a project, using HTML, CSS, JavaScript, and jQuery.

Using this app, a user can drag a mouse across a canvas to make free-form lines.

Compatible with desktop. Not compatible with mobile devices at this time. The app works in browsers compatible with Windows. Apple users, let me know if it works for you, too!


What Else Can This App Do?

  • Paint with a color with an opacity of their choice
  • Use a dropper to select a color on screen
  • Change the pen size
  • Use an eraser
  • Download painting
  • Delete the painting (with no hope of recovery)
